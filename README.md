# ranked-gamev2

This is a Next.js project prepared for deployment to Vercel.

Getting started

1. Install dependencies

```bash
npm install
```

2. Run development server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
```

Deployment

- Push the `main` branch to GitHub (already present at https://github.com/deevij/ranked-gamev2).
- Import the repository on Vercel and select the `main` branch; Vercel auto-detects Next.js. The `vercel.json` included uses the Next.js builder.

Learn more: https://nextjs.org/docs/deployment
